## What is Neru?

**Neru (練る)** — Japanese for _"to refine and master through practice"_ — lets you navigate, click, and scroll anywhere on your screen using only your keyboard. No mouse. No trackpad. No limits.

It's the free, open-source alternative to Homerow, Mouseless, and Wooshy — with zero paywalls, zero subscriptions, and everything configurable down to the last pixel.

## Quick Start

Get up and running in under 2 minutes:

<Steps>
  <Step title="Install via Homebrew">
    ```bash
    brew tap y3owk1n/tap
    brew install --cask y3owk1n/tap/neru
    ```
  </Step>
  
  <Step title="Grant Accessibility Permission">
    Open **System Settings → Privacy & Security → Accessibility** and enable **Neru**
  </Step>
  
  <Step title="Launch and Try">
    ```bash
    open -a Neru
    ```
    
    Press `Cmd+Shift+C` for recursive grid mode, or `Cmd+Shift+Space` for hints mode.
  </Step>
</Steps>

## Works Everywhere

Neru works across native macOS apps, Electron apps, browsers, creative tools, and system UI:

| Category | Applications |
|----------|-------------|
| **Native macOS** | Finder, Safari, Mail, System Settings, Terminal |
| **Electron** | VS Code, Cursor, Slack, Spotify, Obsidian, Discord |
| **Browsers** | Chrome, Firefox, Safari, Arc, Brave, Zen |
| **Creative** | Figma, Adobe Illustrator, Photoshop |
| **System UI** | Menubar, Dock, Mission Control, Notification Center |

<Info>
  Grid mode works universally. Hints mode works where the accessibility tree is exposed — see the [Troubleshooting Guide](/troubleshooting/app-compatibility) for app-specific notes.
</Info>

## Platform Support

<Note>
  Neru is **100% compatible with macOS**. All features are fully functional and stable.
</Note>

<Warning>
  **Linux & Windows** support is planned but requires community contribution. The infrastructure and interfaces are in place, but native implementations for accessibility, event capture, and overlays need development.
</Warning>

See [Cross-Platform Development](/development/cross-platform) for technical details and contribution opportunities.
